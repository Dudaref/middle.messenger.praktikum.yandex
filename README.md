## Установка проекта

Установка зависимостей:

```bash
npm i
```

## Запуск проекта

Сборка проекта и запуск статического сервера на Express, на порту 3000. http://localhost:3000:

```bash
npm run start
```
## Дополнительные команды 

Сборка проекта:

```bash
npm run build
```

Сборка проекта с наблюдением за файлами:

```bash
npm run dev
```
